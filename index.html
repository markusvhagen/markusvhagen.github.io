Authentication system

PHP / JAVASCRIPT

Simple system with login and registration. Details are stored in database, passwords are hashed with bcrypt. When logging in you can see if username is available or not, and a algorithm will decide the strength of your desired password. When logged in you can post a post that everyone could see.

Simplified.js

JAVASCRIPT

Simplifed.js is a lightweight JavaScript-library, which aim to simplify the development in front-end. It has basic functionality as get-methods, changing/getting css, show/hide and scrolling to a specific element. All methods have a time-argument, which makes it easy to customize.
